# Sakila
Web application for MySQL Sakila sample database - standard schema that can be used for examples in books, tutorials, articles, samples, and so forth.

Backend:
- Java 8
- Spring Boot
- Spring Security
- Spring Data
- JPA
- Hibernate
- Jackson
- REST
- JSON 
- MySQL
- Ehcache 
- Tomcat
- Maven

Frontend:
- Angular
- Angular CLI
- TypeScript
- RxJS 
- Angular Material
- npm
